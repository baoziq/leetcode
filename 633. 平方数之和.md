### 633. 平方数之和



给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c 。

 

示例 1：


输入：c = 5
输出：true
解释：1 * 1 + 2 * 2 = 5


示例 2：


输入：c = 3
输出：false




提示：

 * 0 <= c <= 231 - 1

```c++
class Solution {
public:
    bool judgeSquareSum(int c) {
        int left = 0;
        int right = sqrt(c);
        while (left <= right) {
            long long sum = left * left * 1ll + right * right * 1ll;
            if (sum == c) {
                return true;
            } else if (sum > c) {
                right--;
            } else {
                left++;
            }
        }
        return false;
    }
};
```
